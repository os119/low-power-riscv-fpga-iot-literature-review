# Low-Power RISC-V and FPGA-Based SoC Architectures for IoT Devices

This repository includes my literature review on low-power embedded architectures for IoT applications.

I wrote this review to explore how recent research is improving embedded systems in terms of energy efficiency, flexibility, and edge intelligence. The review mainly looks at RISC-V platforms, FPGA-based SoC design, DVFS, and TinyML.

Working on this helped me build a better understanding of current challenges in low-power system design and how hardware and software decisions affect performance and energy consumption in IoT devices.

## Main areas covered
- RISC-V and configurable embedded architectures
- FPGA-based SoC platforms
- Dynamic Voltage and Frequency Scaling (DVFS)
- TinyML on resource-constrained devices
